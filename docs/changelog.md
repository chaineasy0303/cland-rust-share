# 版本变更记录

## v0.1.0 (当前版本) - 2025-10-26

### 新增功能
- **初始版本发布**
  - 基础项目结构搭建
  - 统一错误处理模块 (`error.rs`)
  - 配置管理模块 (`config/`)
  - 加密工具模块 (`crypto/`)
  - 常用工具函数模块 (`utils/`)

### 模块详情

#### 错误处理模块 (`error.rs`)
- 定义统一的 `CommonError` 枚举类型
- 支持配置错误、加密错误、I/O 错误等
- 提供便捷的错误构造方法
- 实现 `thiserror::Error` 特征

#### 配置管理模块 (`config/`)
- `Config` 主配置结构体
- `ConfigBuilder` 构建器模式实现
- 支持环境变量加载 (`from_env`)
- 提供合理的默认配置

#### 加密工具模块 (`crypto/`)
- AES-256 加密/解密函数
- SHA-256 哈希函数
- 安全随机数生成
- XOR 加密/解密（演示用途）

#### 工具函数模块 (`utils/`)
- 字符串工具（空白检查、蛇形命名、截断）
- 时间工具（时间戳、时长格式化）
- 文件系统工具（文件读写、目录创建）
- 验证工具（邮箱、URL、电话验证）
- 集合工具（向量转哈希表、唯一性检查）

### 测试覆盖
- 集成测试目录 (`tests/`)
  - `config_test.rs` - 配置模块测试
  - `crypto_test.rs` - 加密模块测试  
  - `utils_test.rs` - 工具模块测试

### 文档与示例

- `examples/simple_usage.rs` - 基础使用示例
